# Panel de 71 Agentes IA

Un panel interactivo y moderno que presenta 71 agentes de inteligencia artificial especializados en diferentes áreas. Cada agente está diseñado para proporcionar asistencia experta en su campo específico.

## 🚀 Características

- **71 Agentes Especializados**: Desde medicina hasta tecnología, gastronomía y arte
- **Diseño Moderno**: Interfaz atractiva con animaciones suaves y efectos visuales
- **Búsqueda Inteligente**: Encuentra agentes por nombre, especialidad o descripción
- **Filtros por Categoría**: Organiza agentes por área de expertise
- **Responsive Design**: Optimizado para desktop, tablet y móvil
- **Integración con ChatGPT**: Redirección directa al chat con contexto del agente

## 🛠️ Tecnologías Utilizadas

- **React 18**: Framework principal
- **Vite**: Herramienta de desarrollo rápida
- **Tailwind CSS**: Framework de estilos utilitarios
- **Lucide React**: Iconos modernos
- **PostCSS**: Procesamiento de CSS

## 📦 Instalación

1. **Instalar dependencias**:
   ```bash
   npm install
   ```

2. **Ejecutar en modo desarrollo**:
   ```bash
   npm run dev
   ```

3. **Abrir en el navegador**:
   El proyecto se abrirá automáticamente en `http://localhost:3000`

## 🎯 Uso

1. **Explorar Agentes**: Navega por las tarjetas de los 50 agentes disponibles
2. **Buscar**: Utiliza la barra de búsqueda para encontrar agentes específicos
3. **Filtrar**: Selecciona una categoría para ver agentes de un área específica
4. **Iniciar Chat**: Haz clic en "Iniciar Chat" para ser redirigido a ChatGPT con el contexto del agente

## 📱 Categorías de Agentes

- **Salud**: Médicos, nutricionistas, psicólogos
- **Tecnología**: Desarrolladores, ingenieros, especialistas en IA
- **Educación**: Profesores, investigadores, académicos
- **Negocios**: Consultores, marketers, financieros
- **Gastronomía**: Chefs especializados en diferentes cocinas
- **Diseño**: Arquitectos, diseñadores gráficos, urbanistas
- **Arte**: Músicos, fotógrafos, escultores

## 🔧 Scripts Disponibles

- `npm run dev`: Ejecuta el servidor de desarrollo
- `npm run build`: Construye la aplicación para producción
- `npm run preview`: Previsualiza la build de producción

## 🎨 Personalización

El proyecto está diseñado para ser fácilmente personalizable:

- **Agregar nuevos agentes**: Modifica el archivo `src/data/agents.js`
- **Cambiar estilos**: Personaliza los colores en `tailwind.config.js`
- **Modificar categorías**: Actualiza el array de categorías en `agents.js`
